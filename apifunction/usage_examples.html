Master Table 사용 가이드

이 문서는 master_table.py의 핵심 함수 build_master_table(), load_table(), fetch_one() 중심으로 작성되었습니다.

안내: master_table.py는 프로젝트(apifunction 폴더)에 이미 포함되어 있으므로 별도 다운로드 없이 바로 import해서 사용할 수 있습니다.

목차

1) 개요 2) 마스터 테이블 스키마 3) build_master_table() 4) load_table() 5) search_table() 6) fetch_one() 7) 동작 원리 8) 포함 데이터(Source) 9) 권장 사용 패턴

1) 개요

전체 흐름은 아래와 같습니다.

  1. 최초 1회: build_master_table()로 카탈로그 수집/생성
  2. CSV 저장: master.to_csv(..., encoding="utf-8-sig")
  3. 일상 사용: load_table()로 빠르게 로드
  4. 단건 조회: fetch_one()로 source별 실제 API 호출

2) 마스터 테이블 스키마

현재 마스터 테이블의 표준 컬럼은 아래 8개입니다.

컬럼 설명 예시
index 고유 식별자. 보통 source:table_id 형태 kosis:DT_1IN2030
source 데이터 출처 kosis, ecos, worldbank
table_id source 내부의 통계표/지표 ID 102Y004, NGDP_RPCH
table_name 표/지표 이름 본원통화 구성내역...
cycle 주기 코드 Y, M, Q, H, F
start_date 조회 시작값(문자열 기준) 2015, 200001
end_date 조회 종료값(문자열 기준) 2024, 202412
params source별 추가 파라미터(dict) {"style_num":"1", ...}

3) build_master_table()

소스별 카탈로그를 동적으로 수집해 master DataFrame을 생성합니다.

KOSIS 파일 준비 방법:
  1. KOSIS 주제별 통계 페이지 에 접속합니다.
  2. 목록 받기를 누른 뒤, 우측 상단 전체 다운로드로 받은 파일(주제별통계.xls)을 kosis_excel_path에 지정해 사용합니다. 
from master_table import build_master_table

master = build_master_table(
    kosis_excel_path="주제별통계.xls",
    include_sources=("kosis", "ecos", "imf", "enara", "molit", "worldbank"),
    timeout=60,
)

print(master.shape)
print(master.columns.tolist())
print(master["source"].value_counts())

# 수집 중 이슈(예: 특정 source 네트워크 장애)
print(master.attrs.get("build_issues", []))
주의: 전체 build는 오래 걸릴 수 있습니다(네트워크/API 상태 영향 큼). 실무에서는 1회 build 후 CSV 저장 + 이후 load_table 재사용을 권장합니다.

4) load_table()

저장해둔 master.csv(UTF-8 SIG)를 빠르게 로드합니다. params 문자열을 dict로 복원합니다. 

from master_table import load_table

master = load_table("master.csv")   # 기본값도 master.csv
print(master.shape)
print(master.dtypes)
print(type(master.iloc[0]["params"]))  # dict

5) search_table()

master table에서 원하는 row를 빠르게 찾는 검색 함수입니다. 

from master_table import search_table

# 1) 키워드 검색 (index/table_id/table_name 전체 대상)
q1 = search_table(master, query="청년", limit=20)

# 2) source + cycle 조합 검색
q2 = search_table(master, source="kosis", cycle="M", limit=50)

# 3) 정확한 table_id로 검색
q3 = search_table(master, table_id="DT_1IN2030")

# 4) params에 특정 key가 있는 row만
q4 = search_table(master, source="molit", params_key="style_num", limit=30)

# 5) 정규식 검색
q5 = search_table(master, query=r"^worldbank:.*ytil", use_regex=True, limit=10)

display(q1.head())
display(q2.head())
display(q3.head())
display(q4.head())
display(q5.head())

지원 기능

  • query: index/table_id/table_name 통합 검색
  • source, cycle, table_id: 구조화 필터
  • params_key: params(dict) 키 존재 여부 필터
  • use_regex, case_sensitive: 문자열 매칭 옵션
  • limit: 반환 행 수 제한

6) fetch_one()

master의 한 row를 기준으로 source에 맞는 API를 자동 호출해 실제 데이터를 가져옵니다. 

from master_table import fetch_one

# 1) table_id 기반 (source 자동 판별)
df1 = fetch_one(master, dataset_id="DT_1IN2030")

# 2) index 기반 (가장 안전)
df2 = fetch_one(master, dataset_id="worldbank:ytil_some_dfcl_all")

# 3) source를 명시하는 방식도 가능
df3 = fetch_one(master, source="imf", dataset_id="NGDP_RPCH", start_year=2010, end_year=2024)

print(df1.head())
print(df2.head())
print(df3.head())

MOLIT 특이사항

  • 한 번에 최대 5년 제한이 있어 내부적으로 5년씩 자동 분할 후 concat
  • OpenAPI가 406이면 public endpoint fallback 시도

7) 동작 원리

  1. Build 단계: source별 카탈로그 수집 후 통일 스키마로 정규화
  2. Load 단계: CSV 로드 + params dict 복원
  3. Fetch 단계: index/source로 dispatcher 분기 후 source 전용 함수 호출
  4. 에러 처리: source별 API 제약(예: KOSIS 대량조회, MOLIT 5년 제한)을 함수 내부에서 처리

8) 포함 데이터(Source)

  • kosis: KOSIS 엑셀/파라미터 기반 통계표
  • ecos: 한국은행 ECOS 통계표
  • imf: IMF DataMapper 지표
  • enara: e-나라지표 XML OpenAPI 지표/통계표
  • molit: 국토교통부 통계누리
  • worldbank: World Bank Indicator 목록

source별 네트워크/서비스 상태에 따라 일부 source가 빈 결과일 수 있으며, 이 경우 master.attrs["build_issues"]에서 확인할 수 있습니다.

9) 권장 사용 패턴

from master_table import build_master_table, load_table, fetch_one

# 최초 1회 (오래 걸릴 수 있음)
master = build_master_table(
    kosis_excel_path="주제별통계.xls",
    include_sources=("kosis", "ecos", "imf", "enara", "molit", "worldbank"),
)
master.to_csv("master.csv", index=False, encoding="utf-8-sig")

# 이후 반복 사용 (빠름)
master = load_table("master.csv")

# 검색해서 대상 row 확인
from master_table import search_table
cands = search_table(master, source="worldbank", query="youth idle", limit=5)
print(cands[["index", "table_id", "table_name"]])

# 필요한 데이터만 fetch
df = fetch_one(master, dataset_id="worldbank:ytil_some_dfcl_all", start_year=2010, end_year=2024)
print(df.head())
